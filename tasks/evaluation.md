# Evaluation Tasks

## Objective

Evaluate models rigorously and turn the results into the final report.

## Tasks

1. Report macro F1 and weighted F1.
2. Generate confusion matrices and per-class metrics.
3. Compare performance by horizon.
4. Analyze which classes become predictable earliest.
5. Run error analysis on failure cases.
6. Add calibration analysis if time allows.
